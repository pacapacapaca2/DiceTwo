Как играть:
1. Бросайте кубики до 3-х раз за ход.
2. После каждого броска можно сохранить любые кубики, нажав на них.
3. После бросков выберите категорию для записи очков.
4. Каждая категория может быть выбрана только один раз за игру.
5. Цель игры - набрать как можно больше очков.
